Welcome to the documentation for Arangopipe. Arangopipe is a tool to manage machine learning projects. It accomplishes this by recording meta-data from the various activities in the life cycle of your machine learning projects. The documentation provides links to various example notebooks that illustrate how Arangopipe can be used with your projects.
